# 使用操作手冊

## 快速開始
1. 開啟網站後，系統會自動載入最新月份的聽力練習內容
2. 首頁會顯示該月份所有主題的列表，按發布日期由新到舊排序
3. 每個主題卡片顯示：標題、類型（如：國小）、標籤、日期和內容預覽

## 如何尋找想要的練習內容

### 選擇練習月份
1. 點選上方的「選擇月份」下拉選單
2. 選擇您想練習的月份（如：2025年06月）
3. 系統會自動載入該月份的所有內容

### 篩選練習程度
1. 在「選擇程度」下拉選單中選擇適合的程度
2. 可選擇：國小、國中等不同程度
3. 列表會立即更新顯示符合條件的內容

### 依主題分類篩選
1. 點選「選擇標籤」下拉選單
2. 選擇感興趣的主題分類（如：科學、運動、文化等）
3. 系統會篩選出相關主題的練習

### 關鍵字搜尋
1. 在搜尋框中輸入關鍵字
2. 可搜尋內容包括：
   - 主題標題
   - 英文內容
   - 中文翻譯
   - 單字表內容
3. 按下 Enter 鍵或點選搜尋按鈕開始搜尋


## 如何進行聽力練習

### 進入練習頁面
1. 點選任一主題卡片
2. 系統會開啟該主題的詳細練習頁面（全螢幕模式）

### 播放音檔
1. 頁面上方有音訊控制器
2. 點選播放按鈕開始播放
3. 可使用進度條跳到指定位置

### 練習內容分頁說明

#### 📝 內容 (Content) 分頁
**基礎練習步驟：**
1. 先點選播放鈕聽一遍完整內容
2. 逐句練習：點選每段文字前的 ▶️ 按鈕跳到該段落
3. 測試理解：先不看中文，試著理解英文內容
4. 確認理解：點選 🌐 按鈕顯示中文翻譯
5. 再次聆聽：重複播放加深印象

**操作技巧：**
- 🌐 按鈕：顯示/隱藏中文翻譯
- ▶️ 按鈕：跳轉到該段落的時間點
- 建議先聽英文，再看中文確認理解

#### 📚 單字 (Vocabulary) 分頁
**學習步驟：**
1. 閱讀前言了解單字學習重點
2. 逐個學習單字：
   - 看英文單字和解釋
   - 點選 ▶️ 按鈕聽發音
   - 理解用法和例句
3. 閱讀後記加深印象

#### 🎯 測驗 (Quiz) 分頁
**測驗步驟：**
1. 閱讀每道題目
2. 點選題目旁的 ▶️ 按鈕聽相關音檔段落
3. 從選項中選擇答案（每題只能選一個）
4. 完成所有題目後，點選「顯示答案」按鈕
5. 檢視結果：
   - ✅ 綠色：正確答案
   - ❌ 紅色：錯誤答案
6. 檢討錯誤並重新聆聽相關段落

## 個人化功能

### 自動記憶功能
- 系統會自動記住您的篩選偏好（月份、程度、標籤、關鍵字）
- 下次開啟網站時會自動套用上次的設定
- 讓您更快找到想要的練習內容

## 生字簿管理功能

### 📚 我的生字簿
點選上方導覽列的「我的生字簿」即可進入個人生字簿管理頁面。

### 如何加入生字

#### 📖 從文章內容選擇單字
1. **進入文章詳細頁面**：點選任一主題卡片開啟文章
2. **選擇要加入的單字**：
   - 在「內容」或「單字」分頁中，用滑鼠選擇英文單字
   - 選擇後會出現藍色提示框
3. **加入生字簿的方式**：
   - **右鍵選單**：選擇文字後點擊右鍵，選擇「加入生字簿」
   - **雙擊快速加入**：直接雙擊單字即可快速加入
   - **觸控設備**：長按單字約1秒，會出現加入選單

#### ✏️ 編輯生字資訊
1. **填寫單字詳細資訊**：
   - **選擇的單字**：系統自動填入（必填）
   - **英標**：可輸入音標，例如：[ˈæpl]
   - **中文翻譯**：輸入單字的中文意思
   - **熟悉度**：選擇「不熟」、「尚可」或「記住了」（必填）
2. **來源文章**：系統會自動記錄單字來源的文章
3. **點擊「儲存」**：完成單字加入

### 生字簿功能操作

#### 🔍 搜尋與篩選
- **熟悉度篩選**：選擇特定熟悉度等級查看單字
- **關鍵字搜尋**：在搜尋框輸入單字或翻譯進行即時過濾
- **智能排序**：單字會依熟悉度和英文字母順序排列

#### ✏️ 編輯與管理
- **編輯單字**：點擊單字旁的 ✏️ 圖示可重新編輯
- **發音功能**：點擊 🔊 按鈕聽取單字發音
- **查看來源**：點擊來源文章標籤可回到原文章
- **刪除單字**：在編輯模式中可刪除不需要的單字

#### 📋 來源文章管理
- **多來源合併**：同一單字在不同文章出現會自動合併來源
- **來源顯示**：每個單字會顯示所有出現過的文章來源
- **快速跳轉**：點擊來源標籤可直接開啟該文章

### 資料管理功能

#### 💾 匯出生字簿
1. 點擊生字簿頁面右上角的 📥 匯出按鈕
2. 系統會自動下載 JSON 格式的備份檔案
3. 檔案名稱格式：`vocabulary-book-時間戳記.json`

#### 📤 匯入生字簿
1. 點擊 📤 匯入按鈕選擇備份檔案
2. 支援格式：
   - **新格式**：完整的 JSON 匯出檔案（推薦）
   - **舊格式**：簡單的單字陣列格式
3. 匯入後會自動載入並顯示所有單字

#### ⚙️ 儲存設定（僅開發模式）
- **LocalStorage**：適合小量資料，簡單快速
- **IndexedDB**：適合大量資料，功能完整（預設）
- **自動遷移**：切換儲存方式時會自動遷移資料

### 學習建議

#### 📊 熟悉度管理
- **不熟**：剛學習的新單字，需要多次複習
- **尚可**：有印象但不太確定的單字
- **記住了**：完全掌握的單字

#### 🎯 有效學習流程
1. **邊聽邊記**：練習聽力時遇到不會的單字立即加入
2. **定期複習**：使用篩選功能專門複習「不熟」的單字
3. **發音練習**：利用發音功能練習正確讀音
4. **來源回顧**：點擊來源文章重溫單字使用語境
5. **進度追蹤**：隨時更新熟悉度，追蹤學習進度

#### 💡 進階技巧
- **批量學習**：一次加入多個單字，再統一編輯詳細資訊
- **主題分類**：利用來源文章功能按主題分類學習
- **定期備份**：使用匯出功能定期備份學習成果
- **跨設備同步**：使用匯出/匯入功能在不同設備間同步資料

## 整體學習建議

### 初學者建議流程
1. **選擇適當程度**：從「國小」程度內容開始
2. **聽力優先**：先聽一遍完整音檔，不要急著看文字
3. **逐步理解**：再聽一遍，這次可以看英文內容
4. **建立生字簿**：遇到不會的單字立即加入生字簿
5. **逐句練習**：使用時間跳轉功能重複聽取
6. **驗證理解**：最後做測驗檢驗理解程度
7. **定期複習**：回到生字簿複習新學的單字

### 進階練習技巧
1. **挑戰理解**：嘗試不看中文翻譯完成理解
2. **跟讀練習**：重複聽取直到能夠跟讀
3. **詞彙擴充**：利用單字分頁系統性學習詞彙
4. **發音矯正**：使用生字簿的發音功能練習讀音
5. **語境學習**：點擊生字簿的來源文章重溫語境
6. **循序漸進**：測驗全對後再進入下一主題

### 有效學習提醒
- **時間安排**：建議每天練習 15-30 分鐘
- **學習策略**：重複練習比一次長時間練習更有效
- **主題選擇**：善用關鍵字搜尋找到感興趣的主題
- **生字管理**：每週檢視生字簿，提升熟悉度等級
- **資料保護**：定期使用匯出功能備份學習成果
- **複習循環**：定期回顧之前學過的內容，強化記憶

### 長期學習規劃
- **階段提升**：逐步從國小程度進階到國中程度
- **主題多元**：嘗試不同標籤的文章，擴展知識面
- **技能整合**：結合聽力、閱讀、詞彙三方面學習
- **自我評估**：利用生字簿追蹤詞彙量成長
- **持續動機**：設定小目標，如每週學會10個新單字

# 技術資料格式說明

## 月份資料管理
+ 月份選項由 `assets/data/months.json` 檔案管理，包含以下格式：
  ```json
  [
    "202506",
    "202505",
    "202504"
  ]
  ```
+ 月份顯示標籤由程式端自動處理，將 `yyyyMM` 格式轉換為 `yyyy年MM月` 顯示。
+ 各月份的資料數據放置在 `assets/data/` 目錄下，檔名格式為 `yyyyMM.json`，例如 `202506.json`。

## 主題資料格式
+ 每個檔案的內容為一個 JSON 陣列，包含多個主題，每個主題包含以下欄位：
  - `id`: 主題的唯一識別碼，格式為 `yyyyMMdd-序號`。
  - `type`: 主題的類型，例如 "國小"。
  - `title`: 主題的標題。
  - `audio`: 音效檔案的 URL。
  - `tag`: 標籤陣列（可選）。
  - `content`: 一個陣列，包含多個內容項目，每個項目包含：
    - `en`: 英文內容。
    - `tw`: 中文翻譯（可選）。
    - `time`: 音效時間點（可選），支援 "分:秒" 或 "時:分:秒" 格式。
  - `vocabulary`: 單字列表，包含：
    - `preface`: 前言（可選）
    - `content`: 單字內容陣列，每項包含：
      - `text`: 單字說明文字
      - `time`: 音效時間點（可選）
    - `postscript`: 後記（可選）
  - `quiz`: 測驗題目陣列，每題包含：
    - `question`: 題目
    - `options`: 選項陣列
    - `answer`: 正確答案
    - `time`: 音效時間點（可選）

## 標籤管理
+ 標籤選項由 `assets/data/tag.json` 檔案管理，包含所有可用標籤的陣列。
+ 所有 JSON 檔案載入時會附加隨機查詢字串 `?v=timestamp&r=random` 防止瀏覽器快取。

## 日期顯示格式
+ 所有日期顯示均包含星期資訊，格式為 `yyyy-MM-dd(星期)`，例如 `2025-06-15(日)`。
+ 日期格式化功能為共用函數 `formatDateWithWeekday()`，確保整個應用程式的一致性。
+ 星期顯示使用中文：日、一、二、三、四、五、六。

## 搜尋功能範圍
搜尋功能支援以下內容的關鍵字查詢：
- 主題標題 (`title`)
- 主題類型 (`type`) 
- 標籤 (`tag`)
- 英文內容 (`content.en`)
- 中文翻譯 (`content.tw`)
- 單字表內容 (`vocabulary.content.text`)

# 使用操作手冊

## 快速開始
1. 開啟網站後，系統會自動載入最新月份的聽力練習內容
2. 首頁會顯示該月份所有主題的列表，按發布日期由新到舊排序
3. 每個主題卡片顯示：標題、類型（如：國小）、標籤、日期和內容預覽

## 如何尋找想要的練習內容

### 選擇練習月份
1. 點選上方的「選擇月份」下拉選單
2. 選擇您想練習的月份（如：2025年06月）
3. 系統會自動載入該月份的所有內容

### 篩選練習程度
1. 在「選擇程度」下拉選單中選擇適合的程度
2. 可選擇：國小、國中等不同程度
3. 列表會立即更新顯示符合條件的內容

### 依主題分類篩選
1. 點選「選擇標籤」下拉選單
2. 選擇感興趣的主題分類（如：科學、運動、文化等）
3. 系統會篩選出相關主題的練習

### 關鍵字搜尋
1. 在搜尋框中輸入關鍵字
2. 可搜尋內容包括：
   - 主題標題
   - 英文內容
   - 中文翻譯
   - 單字表內容
3. 按下 Enter 鍵或點選搜尋按鈕開始搜尋


## 如何進行聽力練習

### 進入練習頁面
1. 點選任一主題卡片
2. 系統會開啟該主題的詳細練習頁面（全螢幕模式）

### 播放音檔
1. 頁面上方有音訊控制器
2. 點選播放按鈕開始播放
3. 可使用進度條跳到指定位置

### 練習內容分頁說明

#### 📝 內容 (Content) 分頁
**基礎練習步驟：**
1. 先點選播放鈕聽一遍完整內容
2. 逐句練習：點選每段文字前的 ▶️ 按鈕跳到該段落
3. 測試理解：先不看中文，試著理解英文內容
4. 確認理解：點選 🌐 按鈕顯示中文翻譯
5. 再次聆聽：重複播放加深印象

**操作技巧：**
- 🌐 按鈕：顯示/隱藏中文翻譯
- ▶️ 按鈕：跳轉到該段落的時間點
- 建議先聽英文，再看中文確認理解

#### 📚 單字 (Vocabulary) 分頁
**學習步驟：**
1. 閱讀前言了解單字學習重點
2. 逐個學習單字：
   - 看英文單字和解釋
   - 點選 ▶️ 按鈕聽發音
   - 理解用法和例句
3. 閱讀後記加深印象

#### 🎯 測驗 (Quiz) 分頁
**測驗步驟：**
1. 閱讀每道題目
2. 點選題目旁的 ▶️ 按鈕聽相關音檔段落
3. 從選項中選擇答案（每題只能選一個）
4. 完成所有題目後，點選「顯示答案」按鈕
5. 檢視結果：
   - ✅ 綠色：正確答案
   - ❌ 紅色：錯誤答案
6. 檢討錯誤並重新聆聽相關段落

## 個人化功能

### 自動記憶功能
- 系統會自動記住您的篩選偏好（月份、程度、標籤、關鍵字）
- 下次開啟網站時會自動套用上次的設定
- 讓您更快找到想要的練習內容


## 學習建議

### 初學者建議流程
1. 選擇「國小」程度內容開始
2. 先聽一遍完整音檔，不要急著看文字
3. 再聽一遍，這次可以看英文內容
4. 逐句練習，使用時間跳轉功能
5. 最後做測驗檢驗理解程度

### 進階練習技巧
1. 挑戰不看中文翻譯完成理解
2. 重複聽取直到能跟讀
3. 利用單字分頁加強詞彙
4. 測驗全對後再進入下一主題

### 有效學習提醒
- 建議每天練習 15-30 分鐘
- 重複練習比一次長時間練習更有效
- 善用關鍵字搜尋找到感興趣的主題
- 定期回顧之前學過的內容

# 技術資料格式說明

## 月份資料管理
+ 月份選項由 `assets/data/months.json` 檔案管理，包含以下格式：
  ```json
  [
    "202506",
    "202505",
    "202504"
  ]
  ```
+ 月份顯示標籤由程式端自動處理，將 `yyyyMM` 格式轉換為 `yyyy年MM月` 顯示。
+ 各月份的資料數據放置在 `assets/data/` 目錄下，檔名格式為 `yyyyMM.json`，例如 `202506.json`。

## 主題資料格式
+ 每個檔案的內容為一個 JSON 陣列，包含多個主題，每個主題包含以下欄位：
  - `id`: 主題的唯一識別碼，格式為 `yyyyMMdd-序號`。
  - `type`: 主題的類型，例如 "國小"。
  - `title`: 主題的標題。
  - `audio`: 音效檔案的 URL。
  - `tag`: 標籤陣列（可選）。
  - `content`: 一個陣列，包含多個內容項目，每個項目包含：
    - `en`: 英文內容。
    - `tw`: 中文翻譯（可選）。
    - `time`: 音效時間點（可選），支援 "分:秒" 或 "時:分:秒" 格式。
  - `vocabulary`: 單字列表，包含：
    - `preface`: 前言（可選）
    - `content`: 單字內容陣列，每項包含：
      - `text`: 單字說明文字
      - `time`: 音效時間點（可選）
    - `postscript`: 後記（可選）
  - `quiz`: 測驗題目陣列，每題包含：
    - `question`: 題目
    - `options`: 選項陣列
    - `answer`: 正確答案
    - `time`: 音效時間點（可選）

## 標籤管理
+ 標籤選項由 `assets/data/tag.json` 檔案管理，包含所有可用標籤的陣列。
+ 所有 JSON 檔案載入時會附加隨機查詢字串 `?v=timestamp&r=random` 防止瀏覽器快取。

## 日期顯示格式
+ 所有日期顯示均包含星期資訊，格式為 `yyyy-MM-dd(星期)`，例如 `2025-06-15(日)`。
+ 日期格式化功能為共用函數 `formatDateWithWeekday()`，確保整個應用程式的一致性。
+ 星期顯示使用中文：日、一、二、三、四、五、六。

## 搜尋功能範圍
搜尋功能支援以下內容的關鍵字查詢：
- 主題標題 (`title`)
- 主題類型 (`type`) 
- 標籤 (`tag`)
- 英文內容 (`content.en`)
- 中文翻譯 (`content.tw`)
- 單字表內容 (`vocabulary.content.text`)
